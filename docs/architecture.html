Frontend Composables

composables/useFilters.js

Global Filter State

Module-level singleton — refs are shared across all components.

  • selectedPeriod / selectedLocation
  • selectedCategory / selectedStatus
  • getCurrentFilters() → API params
  • resetFilters()
composables/useI18n.js

Internationalisation

Singleton; locale persisted to localStorage. en / ja supported.

  • t(key) — key-based translation
  • translateProductName / translateCustomerName
  • translateWarehouse
  • currentCurrency (USD / JPY auto)
composables/useAuth.js

Auth (Mock)

Always-authenticated mock user. Language-aware display fields.

  • currentUser (computed, locale-aware)
  • isAuthenticated (always true)
  • logout() — alert placeholder
  • getInitials(name)
